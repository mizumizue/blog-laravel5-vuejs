# blog-laravel5-vuejs

## Descritpion

自作アプリの練習兼自分の技術ブログも欲しいという理由で制作。
Laravel製のAPIをVue.ｊｓのフロントページから叩きにいくという構成。
Blog管理画面についても同様。

Laravel - Backend
Vue.js - Frontend

## Features

### Frontend

- Blog記事一覧
- Blog記事ページ
- キーワード、タグによる記事検索

### Backend

- API(記事・タグ・管理情報)
- API認証(JWT)

### CMS

- 記事編集（Markdown）
- タグ追加、編集
- サイト情報管理
